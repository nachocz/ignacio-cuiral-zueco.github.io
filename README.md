## ğŸ”§ Setup & Deployment

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/icz_cv_page.git
   cd icz_cv_page
   ```

2. Open `index.html` in your browser:
   - Double-click the file, or
   - Use a local server: `python -m http.server 8000` then visit `http://localhost:8000`

### Deploy with GitHub Pages

1. **Create a GitHub repository** (if not already done):
   - Go to [github.com/new](https://github.com/new)
   - Name it: `icz_cv_page` or `USERNAME.github.io`
   - Make it public
   - Don't initialize with README (we already have one)

2. **Push your code** (from this folder):
   ```bash
   git init
   git add .
   git commit -m "Initial commit: Add CV website"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/REPO_NAME.git
   git push -u origin main
   ```

3. **Enable GitHub Pages**:
   - Go to your repository on GitHub
   - Navigate to **Settings** â†’ **Pages**
   - Under "Source", select:
     - Branch: `main`
     - Folder: `/ (root)`
   - Click **Save**
   - Wait 1-2 minutes for the site to be published

4. **Your CV is now live!**
   - If repo is `USERNAME.github.io`: `https://USERNAME.github.io`
   - If repo is `icz_cv_page`: `https://USERNAME.github.io/icz_cv_page`
